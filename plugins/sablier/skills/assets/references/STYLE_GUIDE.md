# Sablier Visual Style Guide

Official Sablier brand colors and styles.

## Color Palette

### Primary Gradient (Orange)

| Name          | Hex       | Usage          |
| ------------- | --------- | -------------- |
| Primary Start | `#ff7300` | Gradient start |
| Primary Mid   | `#ff9c00` | Median value   |
| Primary End   | `#ffb800` | Gradient end   |

### Secondary Gradient (Blue)

| Name            | Hex       | Usage               |
| --------------- | --------- | ------------------- |
| Secondary Start | `#003dff` | Gradient start      |
| Secondary Mid   | `#0063ff` | Median value        |
| Secondary End   | `#00b7ff` | Gradient end        |
| Desaturated     | `#266cd9` | Softer blue variant |

### Dark Theme

| Name     | Hex       | Usage                |
| -------- | --------- | -------------------- |
| Dark 000 | `#14161f` | Background (landing) |
| Dark 050 | `#1a1d28` | Slightly lighter     |
| Dark 100 | `#1e212f` | Background (app)     |

**Important:** Never use pure black (`#000000`).

## 3D Asset Style

The Sablier 3D asset style:

- Matte plastic finish (not pure matte, not glossy)
- Soft lighting
- No harsh reflections
- Simple clean geometry
- Smooth rounded edges
- Floating/tilted compositions
- Stylized aesthetic

## Asset Guidelines

- Single object or small composition
- Tilted angle for dynamic feel
- Clean isolation for layer use in Figma
- Output: PNG with dark navy background (remove manually in Figma/Photoshop)

## Notes from Brand Guidelines

1. Noise is applied on gradients to reduce banding
1. Dark theme only (light theme not yet available)
1. Logo should not be modified, rotated, or recolored
